# 📊 Financial Company Performance Dashboard (Microsoft Excel)

## Overview

This project is an interactive financial dashboard built in Microsoft Excel using **simulated data** based on the operations of **JPMorgan Chase & Co.**

The goal of this project was to practice data cleaning, analysis, and dashboard design while working with a realistic banking dataset. I wanted to create a dashboard that makes it easy to understand key financial metrics and business performance through interactive visualizations.

> **Note:** The dataset used in this project is completely simulated and was created for learning and portfolio purposes. It does not represent any real customer or company data.

---

## What I Built

The dashboard provides an overview of the company's financial performance by tracking:

- Total Deposits
- Total Withdrawals
- Total Loans
- Total Customers
- Monthly Deposit Trend
- Monthly Withdrawal Trend
- Monthly Loan Trend
- Deposits vs Withdrawals by Month
- Branch-wise Total Deposits
- Loan Status Analysis
- Active vs Inactive Accounts
- Monthly Transaction Count

Interactive slicers allow users to filter the dashboard by different criteria and explore the data more easily.
